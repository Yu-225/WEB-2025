Вбудовані шрифти

Вбудовані шрифти називають безпечними, бо вони одразу встановленні у систему і однаково відображаються на різних пристроях

Приклад використання шрифту serif:

Квітневе проміння розливається між пагорбами, торкаючись кожного листка, немов ніжний подих весни. Стародавні хмари пливуть понад долинами, несячи у собі мелодію далекого часу. Мандрівник іде стежкою, вслухаючись у шепіт трав і дзвін джерельної води, що нагадує про нескінченний рух життя. У тиші лісу народжуються думки, сплітаючись із ароматом землі та подихом вітру, створюючи гармонію, де слова зникають, а залишається лише відчуття істини.

<p class="f-serif">Текст з шрифтом serif</p>
.f-serif { font-family: serif; }

Локальні шрифти

Локальні шрифти - це шрифти, які зберігаються на комп'ютері або сервері. Їх можна підключити до веб-сторінки за допомогою CSS.

Приклад підключення та використання локальних шрифтів:

@font-face {
    font-family: 'Ubuntu';
    src: url('../fonts/Ubuntu/Ubuntu-Regular.ttf') format('truetype');
}

@font-face {
    font-family: 'RubikWetPaint';
    src: url('../fonts/RubikWetPaint/RubikWetPaint-Regular.ttf') format('truetype');
}

@font-face {
    font-family: 'YuFV1-Regular';
    src: url('../fonts/Yuf/YufV1-Regular.ttf') format('truetype'),
         url('../fonts/Yuf/YufV1-Regular.otf') format('opentype');
}

.f-ubuntu { font-family: 'Ubuntu'; }

.f-RubikWetPaint { font-family: 'RubikWetPaint'; }

.f-YuFV1-Regular { font-family: 'YuFV1-Regular'; }

Ґуґл шрифти

Google Fonts - це безкоштовна бібліотека веб-шрифтів, яку можна використовувати на веб-сайтах. Вона пропонує широкий вибір шрифтів, які можна легко підключити до веб-сторінки за допомогою посилання.

Приклад використання Google Fonts:

Press Start 2P

Підключений через HTML:

<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Press+Start+2P&display=swap" rel="stylesheet">
.press-start-2p-regular {
  font-family: "Press Start 2P", system-ui;
  font-weight: 400;
  font-style: normal;
}

Rubik Glitch

Підключений через CSS:

@import url('https://fonts.googleapis.com/css2?family=Rubik+Glitch&display=swap');
.rubik-glitch-regular {
  font-family: "Rubik Glitch", system-ui;
  font-weight: 400;
  font-style: normal;
}

Іконкові шрифти

Іконкові шрифти - це спеціальні шрифти, які містять іконки замість букв. Вони дозволяють легко додавати іконки до веб-сторінок за допомогою CSS.

Приклад використання іконкових шрифтів (FontAwesome):

Для початку потрібно зареєструватись та створити новий набір, та підключити його до сторінки з допомогою згенерованого скрипта:

<script src="https://kit.fontawesome.com/f452764e5f.js" crossorigin="anonymous"></script>

Після підключення можна використовувати іконки в HTML:

<i class="fas fa-camera"></i>
